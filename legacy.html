Can't See Your Bootable USB?

We're here to help.

    Steps:
  1. Get to BIOS menu


  2. Change boot option to legacy.


  3. After the reboot, go to the BIOS setup


  4. Go to the Boot tab


  5. Set Load legacy option ROM to true


  6. Make sure secure boot mode is set to false


  7. Also, set boot list option to legacy. save and restart in the BIOS menu and you should be aboe to see your usb
    8. Go Back To Homepage
