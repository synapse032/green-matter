Pen Testing Tools

These babies might just save your life one day

  • NMAP: Network mapper. Gives details on target. Possible vulnerability spotter.

  • METASPLOIT: Exploit tool. FOSS, but not quite as large of an exploit library as core impact \/\/\/

  • CORE IMPACT: Exploit tool. HUGE library, constantly changing. also at least 30,000

  • WIRESHARK: Traffic monitor. Good for lulz in coffee shops and public places. FOSS. Also allows packet inspection. good for finding fellow hackers, possible adversaries, etc. etc.

  • SQLMAP: Looks for vulns in sql injection. easy to use. FOSS and command line interface

  • IceIX: Administrator and info gatherer. Allows collection of CC data
